
# 微型图书管理系统

## 技术栈：

SpringBoot，
Vue CLI3，
element UI

## 用途

简单的图书管理系统

## 安装

需要一个叫做book的数据库，sql我已经上传在根目录，
自行导入，修改application.yml中的数据库连接配置。

克隆后直接运行BookApplication，访问localhost:9999

## 代码

book_java：SpringBoot相关代码，结构简单易懂。

book_vue：Vue相关代码都在这里，需要你执行yarn install
这个命令是安装所需要的依赖。
